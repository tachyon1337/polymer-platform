# Polymer Platform

UMD build of the Polymer platform. The platform includes

* Custom Elements
* HTML Imports
* Mutation Observers
* Weak Map
* Shadow DOM

## Current Version
v0.3.5

## Usage

### Browserify

```
var platform = require('polymer-platform')
```

### Stand-alone

```
<script src="dist/platform.js"></script>
```

## Installation

```bash
npm install polymer-platform
```

## Build

### Development

```bash
gulp build
```

### Minified
```bash
gulp minify
```
